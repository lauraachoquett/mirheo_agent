# Artificial Bacterial Flagella (ABF)

Simulate single ABFs in a periodic box.
Different geometries of the ABFs are stored in `data`.

## physical parameters:

	Re << 1
	Ma << 1
	eta = 1e-3 Pas
	magnetic field Magnitude: ~ 2mT
	length of body: ~10um
	critical frequency: ~30Hz
